Политика конфиденциальности

Дата последнего обновления: 4 января 2026 г.

1. Общие положения

Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты персональных данных пользователей мобильного приложения BoxArt Lab (далее — «Приложение»), разработанного для платформы Android.

Используя Приложение, вы соглашаетесь с условиями настоящей Политики. Если вы не согласны с условиями Политики, пожалуйста, не используйте Приложение.

2. Оператор персональных данных

Оператором персональных данных является разработчик Приложения. Для связи с оператором используйте контактную информацию, указанную в разделе «Контакты» настоящей Политики.

3. Какие данные мы собираем

Приложение собирает следующие категории данных:

3.1. Данные, собираемые автоматически:

  • Технические данные устройства: модель устройства, версия операционной системы Android, идентификатор рекламы (Advertising ID), информация о подключении к сети (IP-адрес, тип подключения)
  • Данные для push-уведомлений: токен устройства для отправки push-уведомлений через RuStore Push SDK
  • Данные использования: информация о взаимодействии с Приложением (открытие экранов, использование функций)

3.2. Данные, собираемые рекламными SDK:

Приложение использует следующие рекламные SDK, которые могут собирать данные для показа персонализированной рекламы:

  • Yandex Mobile Ads SDK — для показа рекламных объявлений
  • VK Ads SDK (при использовании) — для показа рекламных объявлений

Рекламные SDK могут собирать следующие данные:

  • Идентификатор рекламы (Advertising ID)
  • Примерное местоположение (на основе IP-адреса или данных сети Wi-Fi)
  • Технические характеристики устройства
  • Данные о взаимодействии с рекламой

Важно: Вы можете отключить персонализированную рекламу в настройках вашего устройства Android (Настройки → Google → Реклама → Отключить персонализацию рекламы).

3.3. Данные платежей:

При оформлении подписки через YooKassa обработка платежных данных осуществляется непосредственно платежным сервисом YooKassa в соответствии с их политикой конфиденциальности. Мы не получаем и не храним данные ваших банковских карт.

4. Цели обработки персональных данных

Мы обрабатываем персональные данные для следующих целей:

  • Обеспечение работы Приложения и его функций
  • Показ рекламных объявлений (в том числе персонализированных)
  • Отправка push-уведомлений
  • Обработка подписок и платежей
  • Улучшение работы Приложения и исправление ошибок
  • Соблюдение требований законодательства

5. Правовые основания обработки

Обработка персональных данных осуществляется на основании:

  • Согласия субъекта персональных данных (статья 6 Федерального закона № 152-ФЗ «О персональных данных»)
  • Необходимости для исполнения договора, стороной которого является субъект персональных данных
  • Соблюдения требований законодательства Российской Федерации

6. Передача данных третьим лицам

Мы можем передавать данные следующим третьим лицам:

  • Yandex — для показа рекламы через Yandex Mobile Ads SDK
  • VK (ВКонтакте) — для показа рекламы через VK Ads SDK (при использовании)
  • RuStore — для отправки push-уведомлений через RuStore Push SDK
  • YooKassa — для обработки платежей при оформлении подписки
  • Cloudflare — для хранения push-токенов устройств (через Cloudflare Workers)

Все третьи лица обязаны соблюдать требования законодательства о защите персональных данных и использовать данные только в целях, указанных в их собственных политиках конфиденциальности.

7. Хранение данных

Персональные данные хранятся в течение срока, необходимого для достижения целей обработки, или в течение срока, установленного законодательством Российской Федерации.

Push-токены устройств хранятся на серверах Cloudflare до момента отзыва согласия на обработку персональных данных или удаления Приложения.

8. Меры защиты данных

Мы применяем технические и организационные меры для защиты персональных данных от неправомерного доступа, уничтожения, изменения, блокирования, копирования, предоставления, распространения, а также от иных неправомерных действий.

9. Права пользователей

В соответствии с Федеральным законом № 152-ФЗ «О персональных данных», вы имеете право:

  • Получать информацию о ваших персональных данных, которые мы обрабатываем
  • Требовать уточнения, блокирования или уничтожения персональных данных
  • Отозвать согласие на обработку персональных данных
  • Обжаловать действия или бездействие оператора в уполномоченном органе по защите прав субъектов персональных данных или в судебном порядке

Для реализации ваших прав свяжитесь с нами по контактам, указанным в разделе «Контакты».

10. Использование рекламы

Приложение содержит рекламные объявления, которые могут показываться:

  • При открытии Приложения (App Open Ad)
  • В виде баннеров (Banner Ad)
  • В виде видео за вознаграждение (Rewarded Video Ad)

Реклама может быть персонализированной на основе ваших интересов и данных об использовании устройств. Вы можете отключить персонализацию рекламы в настройках вашего устройства Android.

При оформлении подписки реклама отключается на срок действия подписки.

11. Push-уведомления

Приложение может отправлять push-уведомления для информирования о важных событиях и обновлениях. Вы можете отключить push-уведомления в настройках Приложения или в настройках вашего устройства Android.

Для отправки push-уведомлений используется RuStore Push SDK. Push-токен вашего устройства передается на серверы Cloudflare для хранения и последующей отправки уведомлений.

12. Подписка и платежи

Приложение предлагает возможность оформления подписки для отключения рекламы и доступа к дополнительным функциям. Обработка платежей осуществляется через платежный сервис YooKassa.

При оформлении подписки мы получаем только информацию о статусе подписки (активна/неактивна) и сроке её действия. Данные банковских карт обрабатываются исключительно YooKassa и не передаются нам.

13. Изменения в Политике конфиденциальности

Мы оставляем за собой право вносить изменения в настоящую Политику конфиденциальности. О существенных изменениях мы будем уведомлять вас через Приложение или по электронной почте.

Дата последнего обновления Политики указывается в начале документа. Продолжая использовать Приложение после внесения изменений, вы соглашаетесь с новой версией Политики.
